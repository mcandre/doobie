```console
$ . lib/doobie.sh

$ doobie "Zoinks!"
```

# REQUIREMENTS

* a POSIX compatible shell, e.g. `/bin/sh`, bash, zsh, ksh, dash, posh, ash, Git Bash, bash on Ubuntu on Windows, MinGW, MSYS, or similar.
